# First Class Sensations - Icon Configuration Instructions
# Configuración de iconos para First Class Sensations Ibiza

## 1) Upload Instructions
Sube todos los archivos de iconos a la carpeta public/ de tu proyecto Next.js

## 2) HTML Code Integration
Inserta el siguiente código HTML entre las etiquetas <head> </head>:

<!-- Apple Touch Icons (Optimizado - Solo icono principal) -->
<link rel="apple-touch-icon" sizes="180x180" href="/apple-icon-180x180.png">

<!-- Android Icons -->
<link rel="icon" type="image/png" sizes="192x192" href="/android-icon-192x192.png">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="96x96" href="/favicon-96x96.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png">

<!-- Web App Manifest -->
<link rel="manifest" href="/manifest.json">

<!-- Microsoft Tiles (Optimizado - Solo icono principal) -->
<meta name="msapplication-TileColor" content="#2563eb">
<meta name="msapplication-TileImage" content="/ms-icon-144x144.png">
<meta name="theme-color" content="#ffffff">

## 3) Notas de Optimización
Se han eliminado iconos redundantes para mejorar el rendimiento:
- Solo se mantiene el Apple Touch Icon principal (180x180)
- Solo se mantiene el Microsoft Tile principal (144x144)
- Se conservan todos los favicones esenciales para desktop

## 4) Verification
Verifica que todos los iconos se cargan correctamente en:
- iOS Safari (Apple Touch Icons)
- Android Chrome (Manifest)
- Windows Tiles (BrowserConfig)
- Desktop browsers (Favicon)
